# 🌤️ Aplicativo de Previsão do Tempo em Python

Um aplicativo com interface gráfica desenvolvido em Python que consome dados meteorológicos em tempo real de uma API externa. O projeto foi criado para explorar conceitos de integração web, requisições HTTP, manipulação de arquivos JSON e exibição de imagens dinâmicas.

## 🚀 Funcionalidades
- **Consulta em Tempo Real:** Conecta-se à API do OpenWeatherMap para buscar o clima atual de qualquer cidade do mundo.
- **Tradução Automática:** Exibe a descrição do clima e a temperatura em graus Celsius (°C).
- **Ícones Dinâmicos:** Baixa e exibe na tela o ícone correspondente à condição climática (sol, chuva, nuvens, etc.).
- **Tratamento de Erros:** Alertas amigáveis caso a cidade não seja encontrada ou ocorra falha de conexão.

## 🛠️ Tecnologias e Bibliotecas Utilizadas
- **Python** (Linguagem principal)
- **Tkinter** (Para a construção da interface gráfica)
- **Requests** (Para fazer as requisições HTTP e conversar com a API)
- **Pillow / PIL** (Para processar e exibir as imagens baixadas da internet)
